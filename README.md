# Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/Silas-N-Schlax/rm-prework-odin-hangman.git
```

Navigate to the project directory:

```bash
cd rm-prework-odin-hangman
```

```bash
bundle install
```

Run `ruby main.rb` in your terminal.
